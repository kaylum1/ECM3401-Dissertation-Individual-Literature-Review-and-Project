Understanding the Dashboard

1. Score Banner

At the top, you will see the Score Banner. It indicates the overall security and privacy score for the current website. If it says "Not changed", it means the score hasn't changed recently.

2. Website URL

Just below the score, the URL of the site you're scanning is displayed. This helps you know which page is currently being analyzed.

3. Overall Score Circle

The large colored circle shows a numerical score (e.g., 10 out of 10). - Green color indicates a good score.
- Yellow or red color (if it appears) indicates issues needing attention.

4. Scan Failure Summary

Underneath the score, a message like "1 out of 22 scans failed" appears. This tells you how many issues were found in the scan. If there are failures, it's a good idea to investigate and fix them.

5. Navigation Buttons

At the bottom, you’ll find a series of Navigation Buttons:

  • Security — View all security scan results.
  • Privacy — View all privacy scan results.
  • Metrics — View performance-related metrics.
  • Logs — View logs of previous scans.
  • Settings — Adjust extension settings.
  • Edu — Educational content about different scans.
  • Help — (this page) Learn how the extension works.
Clicking any button will navigate you to the corresponding page for more details.

Tips for Best Use

  • Always check both Security and Privacy sections after a scan.
  • If you see a low score, explore the details provided under each scan type.
  • Use the Education section ("Edu") to better understand what each scan checks for and why it's important.
  • Visit Settings to customize how aggressive the scans should be.
